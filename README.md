# Simple Login Form with Social Integration

Um formulário de login moderno, intuitivo e responsivo desenvolvido com HTML5 e CSS3. A interface organiza de forma limpa opções de login social no topo e campos tradicionais de autenticação logo abaixo, utilizando um esquema de cores focado em tons roxos vibrantes.

## Tecnologias Utilizadas

* HTML5: Estruturação dos elementos de formulário, campos de entrada e botões de autenticação externa.
* CSS3: Estilização completa, uso de bordas arredondadas, sombras projetadas e definição de cores institucionais.
* Flexbox: Alinhamento centralizado do card principal e distribuição em grade para os botões sociais superiores.

## Funcionalidades e Técnicas Aplicadas

* Autenticação Social no Topo: Botões destacados colocados na parte superior para login rápido via Facebook e Twitter, utilizando suas respectivas identidades visuais de cores.
* Inputs com Ícones Integrados: Campos de preenchimento para e-mail/telefone e senha com pequenos ícones identificadores em tons de cinza no canto esquerdo.
* Opções de Acesso Inline: Caixa de seleção para a função "Remember me" alinhada horizontalmente com o link "Forgot password?".
* Botão de Ação Sólido: Botão principal "Login" estilizado com cantos levemente arredondados e preenchimento roxo de alto contraste.
* Rodapé de Cadastro: Chamada de texto minimalista na base do cartão para direcionar novos usuários para a página de registro ("Signup now").
